# White-Goods
Find Your Goods
This ASP.NET MVC application helps you locate specific goods from a database using their common and unique features.

# Features
-Database Search: Query the database for goods based on their attributes. 

-Feature Identification: Utilize both common and unique features to find the desired items.

-User-Friendly Interface: Simple and intuitive forms for easy searching.

# Usage
-Search for Goods: Enter features in the search form and submit.

-Filter Results: Use additional features to refine your search.
